SASS commands in NPM

NPM install SASS -g (global)
sass input.scss style.css (maps the css files to sass file for preprocessing)
sass --watch input.scss style.css (recompiles the CSS each time you save the sass file)
sass --watch --app/sass:public/stylesheets (watch entire folders seperating them with colon)

SASS should watch all files in app/sass and compile to public/stylesheets folder.

Minify CSS

sass --watch input.scss style.css --style compressed
This minifies the .css file so it can run faster
You can also minify "plain" css this way: sass --watch a.css:a.min.css --style compressed --scss
